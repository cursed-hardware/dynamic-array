# Динамический Массив

Динамический массив — это структура данных, которая может изменять свой размер в процессе выполнения программы. В отличие от статических массивов, размеры которых определяются при компиляции, динамические массивы позволяют программистам управлять памятью более эффективно.

## Основные характеристики:
- Изменяемый размер: Массива может расти или сокращаться в зависимости от требований.
- Эффективность использования памяти: Память выделяется по мере необходимости.
- Произвольный доступ: Как и в статических массивах, элементы доступны по индексу.

## Применение:
Динамические массивы используются в различных задачах, таких как реализация стеков, очередей и других абстрактных структур данных.

## Примеры использования:
Например, динамические массивы обычно применяются в языках программирования, таких как C++, Java, и Python для создания изменяемых коллекций объектов.
